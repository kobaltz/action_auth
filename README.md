# ActionAuth
This is a placeholder for the ActionAuth gem.  It is not yet ready for use.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "action_auth"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install action_auth
```
